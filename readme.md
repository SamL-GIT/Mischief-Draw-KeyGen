# Mischief Draw Key Generator

### What is this ?

This is a simple key generator script for Mischief draw, It's mainly intended as a  way to generate keys when official ones are no longer available.

### Why Does This Exist?

Because Mischief is discontinued 

### How To Use it?

1.  **Install Node.js**
2.  **Copy and Run:** Copy the `key_gen.js` file to a folder, open an terminal inside the folder and then run `node key_gen.js` , enter any email, press enter again and it will give you the corresponding product key.
